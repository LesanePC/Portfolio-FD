# 🚀 Portfolio-FD — персональное портфолио Frontend-разработчика

Добро пожаловать в мой проект **Portfolio-FD** — это мой личный сайт-портфолио, где представлены мои навыки, лучшие работы и контакты для связи.  
Проект создан с нуля с упором на чистую вёрстку, аккуратный UI и адаптивность.

---

## 🎯 Цели проекта

- Представить мои профессиональные и учебные проекты  
- Создать современное и удобное портфолио для работодателей и клиентов  
- Продемонстрировать владение HTML, CSS, JavaScript  
- Показать умение работать с Git и GitHub Pages  
- Подготовить основу, которую можно постоянно улучшать и развивать

---

## 🛠 Использованные технологии

### **Frontend**
- 💡 HTML5 — семантическая разметка  
- 🎨 CSS3 — Flexbox, Grid, адаптивность, медиазапросы  
- ⚡ JavaScript — интерактивность, обработчики событий  
- ✨ CSS-анимации и плавные переходы  

### **Инструменты и окружение**
- 🧰 VS Code  
- 🔧 ESLint (конфиг в проекте)  
- 📁 Git / GitHub  
- 🌍 GitHub Pages (развёртывание)

## ⬇️ Установка и запуск локально

```bash

# Клонировать репозиторий
git clone https://github.com/LesanePC/Portfolio-FD.git

# Перейти в папку проекта
cd Portfolio-FD

# Открыть в браузере (достаточно просто открыть index.html)
open index.html  # для macOS
# или
start index.html # для Windows

```
